Curious, have you tried the latest stable version?  I use 1.2.10 on my 
Debian boxen running Sarge.  I've never had any problems with that. 
Also, does this machine face the Internet?  If so, I wouldn't recommend 
using an RC release unless you have a really good reason, especially if 
the microversion is a "0".

Have you checked the logs to see if there's anything amiss?  ProFTPD 
seems to also be VERY picky about PAM.  PAM has to be working properly 
in order to work.  When I was running Slackware, I had to install PAM in 
order to get ProFTPD working in the first place (Slackware does not use 
PAM for "security reasons").
