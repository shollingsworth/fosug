Message-Id: <f73f9abd05030213321db4dc68@mail.gmail.com> 
I thought I'd share with everyone my first "Production Server
Recovery" using Linux!

Yesterday we shut down our IIS webserver for regular Maintenance and
upon reboot received BSOD (inaccessible boot device) I wrestled with
the RAID card for hours just trying to get it to see the windows
partition with no luck. So I booted with knoppix and it loaded the
modules for the RAID card as well as allowing me to access and mount
the partitions on the array. So I mirrored the paritions (including
MBR) from the array to an IDE HD using dd. The server rebooted
perfectly!

Hurray for Linux!
