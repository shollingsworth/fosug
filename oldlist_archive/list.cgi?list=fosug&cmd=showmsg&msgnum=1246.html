> This is NOT a thumb drive(at least not in the traditional sense). Most
> people consider a thumb drive to have a USB interface(which this one
> has) with flash memory(this one is a hard drive). This is more like a
> external hard drive which gets it's power from USB(not really a good
> idea for an external hard drive, imho).


If the device is a laptop harddrive or even better yet a micro drive.
The power drain is minimal to run them.  I would not worry about
running them off a usb.  They will significantly reduce the battery
life on a laptop tho.  3.5 (full size) harddrives can ACTUALLY be
powered off usb. however they push it to it's limit.  Cheap hardware
might burnout.  So you will RARELY see a full size harddrive run off
usb. IF you have one. Get the external adaptor. or invest in a new
box. ($20-30 for nice external case).
