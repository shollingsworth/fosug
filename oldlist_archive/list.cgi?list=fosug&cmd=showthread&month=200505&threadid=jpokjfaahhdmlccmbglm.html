I've looked into using it.  I bought an ibook to
mess with, so I was checking into OS X and other
ppc unix's.  GNU-Darwin is an opensource version
of Apple's Darwin.  They both sport x86 versions,
only Apple's is not free (its still open source 
though).  

   GNU-Darwin: http://gnu-darwin.sourceforge.net/index.php
   their beta x86 iso: http://prdownloads.sourceforge.net/gnu-darwin/gnu-darwin-cd-beta2.5-x86.iso.gz

It would be interesting to see what it provides.
Apple's Darwin has a ports system (darwin-ports),
so its FreeBSD like.  The main reason I havn't
installed is lack of time and not wanting to run
my x86_64 as 32 bits :)
