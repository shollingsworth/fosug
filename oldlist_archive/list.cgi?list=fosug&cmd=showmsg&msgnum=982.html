Ronnie,

Linux users are frequently true evangelists in what distribution of windows
manager, KDE, Gnome, CLI, ect. ect., they use.  To some extent opinionated.
Open source is a lot more than Linux.  Its the free applications that run on
virtually all platforms and OS's.  The thing to overcome when going to Linux
isn't that its difficult -- it that its different than Windows.  Humans are
so change adversest.  Using the basics is actually easier than Windows.  The
danger comes when you become infatuated with the power underneath and get
into areas that are difficult.  IMHO, the trick to Linux success is small
steps and stick with the GUI for most activities.  One can't help but end up
at the command line because everyone tells them too.  My recommendation is
to learn the command line first by examining things, not by changing them.
Changing will come over time as you understand the foundation and modularity
of Linux.  Even Windows users can benefit from open source applications that
also run on Windows.  In fact that is the path I take my with those that I
support.  Once you're familiar with the applications, a change to Linux is
simple for Desktop users.  As to geeks like me, the easy way is rarely fun
enough.

George
