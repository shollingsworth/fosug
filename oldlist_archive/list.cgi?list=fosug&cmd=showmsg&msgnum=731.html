On 2005.11.03 08:17:27 -0800, Drew Baker wrote:
> Does anyone know a full featured out of box. Free distrubustion email
> server? Single or Multi Domains, Spam filtering, web management? Looking for
> some very quick solid distro that will require little or no setup to use in
> locations were groupwise/exchange will not be used. (no comments on exhcange
> haters. hehehe). Currently we have been using custom gentoo systems with
> vpop and custom scipts but they take way too long to setup.

If I were wanting this to be repeatable, I would strive to make something
that's based on a package distro or move to a BSD that has an automated
installer.

This gives you the base install and the basic packages you need plus a method
to automate the install of the other packages you're interested in.

There are several pop toaster sites out there that give a checklist of
installation options.

Another option is to get a golden machine installed and configured for generic
set of packages and scripts and then just dd the entire thing to another disc
in the machine.  Hopefully, this will be kept under a single cd or dvd in size
and then you can dd it from a network location via ssh using something like
bbc.

Just some thoughts.
