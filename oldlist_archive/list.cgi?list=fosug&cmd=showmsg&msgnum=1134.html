Well I had 3 things to consider. 1. Size, 2. Price, 3. Linux Compatability.
 I ended up with a Averatech 3700
http://www.averatec.com/notebooks/3700series.htm

The key buying point was the fact it was $750 (not including an surprise
$150 mail in).
My "old" Laptop which I am selling.  was a Toshiba Satalite Pro. NICE
laptop. 15" screen XGA. 1gb ram. 2.7ghz.  Wifi, cdburner, 60gb hdd, upto 4
hours battery time. this guy was more then fast enought to run everything.
ran Gentoo on it, with vmware in background and it flew.  Only problem was
it's not anything close to compact. The new one, actually is a bit slower.

AMD64 3000+ (about 1.8ghz)
512mb (upgradable to 2gb by voiding warranty, you have to dismantle laptop)
80gb hdd.
DVD burner (specs don't say, but i'm pretty sure it is NOT dual layer)
12" screen
nice keyboard, not too tinny.  It has a touchpad (I hate erasers too).
3 usb 2.0 ports.
10/100 Via Ethernet
Modem. (via chipset ac97, there is support in there, but honestly I probably
will never care enought to try and set it up)
headphone and Mic jacks (sound works under linux, Via ac97 audio dirt easy
setup)
VGA port
firewire
1 SD/MM/sony memory stick slot (I have not tested under linux yet)
1 pcmcia slot (tested works fine)
802.11a/g wifi (native linux driveers RA2500, and yes you can snoop with it,
already decoded my first wep key)
weights about 4lbs
Manufature claims 4 hours battery life (so in reality you should halve it)

Both times i've had it under battery power, I get about 2.5 hours, with a
fair amount of cpu utilization.  Lots of compiling, etc.  I just found the
power conserving features of the kernel today and enabled them.  They are
suppost to power down the cpu/pcmcia/etc. when laptop is idle and
automaticaly power it back to full speed when needed. So  far it work, we
will see how well battery lasts with it on.


Quick google search gave me a fairly good feeling of it's linux
compatability.  I brought it home, and figured I would wrestle  with it for
a week and see if I could get gentoo on it, return it if I had too many
problems.  Turned out I had my gentoo system running in less then an hour.
(minus wifi, and 3d accleration)  I was able to just stage 4 my amd desktop.
Dump it on my laptop. and recompile kernel for laptop hardware.  Another day
of playing I got the wifi working, was easy once I found out the exact
chipset, took about 20 minutes to install and compile.

Drawback on laptop.  The Video has nill for 3d accleration. It is a S3
UniChrome.  Looks great in 2d. but don't expect to be playing 3d games with
it. (I don't game on my laptop so not a big deal, but I would have liked to
have nice gl screen savers) My X is running in VESA mode right now.  I have
found linux binary drivers from S3 that I can install to improve
performance, but I have yet to try this. I will update when I get around to
it.
