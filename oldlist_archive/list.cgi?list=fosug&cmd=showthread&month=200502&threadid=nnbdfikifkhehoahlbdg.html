First of all the hardware. AMD64 3000, 1GB Ram, 200 and 160 GB HD's. 

I first booted using the x86 2004.3 Universal CD and Gentoo
autodetected all hardware. But when I booted using the AMD64 Universal
CD, Gentoo failed to detect my NIC and sATA hard drives. By using to
noapic boot option, Gentoo saw my sATA HD's but I still had to
manually load the modules for my NIC. No big deal, as the rest of the
installation went fine. I have to say that emerging xorg and kde was a
blast. Compile times were awesome. After the KDE install, I noticed
that some of KDE's menu items are not available and openoffice is
masked. Through a lot of reading, learned that OpenOffice is currently
only available in 32-bit binary form.

I know that Suse 9.1 and earlier offer AMD64 support, does anyone know
if suse offers openoffice on the AMD64 platform? If so how did they do
it?

Any help or suggestions would be greatly appreciated. :-)

Thanks,

-Mike
