This is NOT a thumb drive(at least not in the traditional sense). Most
people consider a thumb drive to have a USB interface(which this one
has) with flash memory(this one is a hard drive). This is more like a
external hard drive which gets it's power from USB(not really a good
idea for an external hard drive, imho).

If you want an external USB hard drive like this, you could build it
yourself (external USB hard drive enclosure($30) + 80 GB hd ($50)) for
much cheaper.
