My first Linux was Linspire.  My biggest problem with it is that they make
small changes to core libraries that make them unique.  There fore if you
install from another source the libraries might be updated to "normal" Linux
breaking Linspire supplied versions of other applications.  This makes it
impossible to use the latest stable releases of software like Firefox or
OpenOffice.  They don't seem to be in a hurry to make those new releases
available.  They have effectively created a quasi proprietary environment
from open source.  I changed to SuSE which installs just as easily and
allows me to participate in the open source community instead of the
Linspire community.  I will not recommend it to anyone.  Linspire stands
alone in the open source world.  IMHO, its the Microsoft of the Linux world.

George
