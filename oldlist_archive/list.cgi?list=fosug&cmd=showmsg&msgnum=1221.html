Ronnie,

Sorry to here that the 'desktop' installer did not work well for you.
What you have now with the 'Server' install is a system that is not
set up to be a graphical workstation, but server for doing things like
serving web pages, databases, etc.

All is not lost however,  the following command should get you a KDE
desktop and windowing environment:

sudo apt-get install kubuntu-desktop

It will download all the KDE files and put them on your box.  Basicly,
you will be running Kubuntu (which is ubuntu with the KDE desktop
instead of the gnome desktop).  If you don't want to run KDE and would
prefer to run gnome, please let me know and we can figure out which
packages you will require
