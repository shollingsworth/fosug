I'll be 100% honest:  I'm cheap, and thus I prefer the 
free-as-in-beer-AND-speech distros, like Debian and Ubuntu, though I

not affect a straight-up Kubuntu install (though I need to try an 
apt-get upgrade once I have Kubuntu fully installed on Kurumi, the box 
I'm going to use as my actual screening box).
