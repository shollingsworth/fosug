Ronnie,

User powers are controlled by owner, group and the permissions set for each
file.  The permutations are much greater than for Windows.  You could for
example give a user a subset or single administrative power.  Just add
another user, they will by default not have administrative powers.  I'm
running Ubuntu, Gnome instead of KDE, and there isn't administrator log in
per se.  A user, by default the first configured, must enter their password
for each administrator function they attempt to perform.
