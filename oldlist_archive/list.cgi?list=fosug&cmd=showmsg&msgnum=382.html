Hello All,

Just thought I'd let you know what the meeting topics were for the
meeting on the 21st.

====================
Beginning Session:
====================
Because of the overwhelming response on the list from beginning
members asking for package management, and gui interfaces
specifically, I figured I'd volunteer to present on the Ubuntu linux
distro.

http://www.ubuntulinux.org

I'll be going over GUI and only GUI-- no command line, I promise. All
distro's are a little bit different, but you'll most likely take
something away due to the fact that they are also the same in many
ways.

No command line is what I'll focus on. Hopefully we can get some folks
hooked, and using linux as a primary desktop.

I've never done much with this particular distrobution, but some
people I trust have had some great things to say, so I'll be learning
a thing or two as well =)

====================
Advanced Session:
====================
Our very own guru programmer, currently a student as Fresno State,
Elliot Johnson will be going over TCP/IP and sockets as it related to
the FOSS programming Envoronment. This is going to be a great
presentation.
Elliot enjoys programming in listp
http://www.apl.jhu.edu/~hall/lisp.html
and C
I'm sure many of the network/programming guys will get a great deal
out of this presentation. I'm looking forward to it!
