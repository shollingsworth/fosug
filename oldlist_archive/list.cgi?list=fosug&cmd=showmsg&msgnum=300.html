On Fri, Mar 25, 2005 at 09:01:33PM -0800, George Fragos wrote:
> I have some donated PCs running Windows 2000 for Fresno's Emergency
> Operations Center.  I plan to run the EOC on 100% open source.  I'm
> trying to get SuSE 9.2 installed but the Windows 2000 won't let me
> boot from a CD and won't let me start the BIOS.  I don't need to dual
> boot -- Linux all the way.  Are there some words of wisdom anybody can
> share with me.
> -- 


Has nothing to do with Windows 2000. It's at the BIOS level. You need to get
into the BIOS to set it to boot from the CD. The usual keys to try, during
POST are F1, ESC or DEL keys. I think I ran across an F10 one time. Most of
the white box board will tell you when it's booting to press a key.

If those don't work, please post the Make, and any other info of the PC or the
motherboard if it's a white box system.
