To me, as an unpaid professional bum, open source gives me a chance to
learn more about OS's, mail servers, etc.  whereas if open source did
not exist, I could not afford to learn from using the commerical OS's.

On the other hand, open source is not all about free software.  It's
about choices.  If you don't like a software, then use another one
that does the same thing.  Else, write your own using pre-existing
open source apps.  It also gives everyone a chance to look over your
source code and audit it (e.g., OpenBSD).  Another key point in open
source is that you can participate in the open source project (try
that with a commerical OS! hehe).
