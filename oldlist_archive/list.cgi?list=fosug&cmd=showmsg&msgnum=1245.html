Hi Ronnie,

Synaptic is basicly a GUI on top of the apt package management system,
and relies on the underlying structure of that system.  What's nice
about apt is that you can define multiple repositories of packages,
and both Ubuntu and Debian (upon which Ubuntu is based) have various
repositories to choose from.

In Ubuntu (and therefore Kubuntu) there are 3 main repositories: Main,
Universe, and Multiverse  (for an overview of what exists in each
repository please see: http://www.ubuntu.com/ubuntu/components <--
Good Reading) as well as a Security repository where updated packages
to fix security issues reside.

For reasons hopefully made clear in the above link, Ubuntu comes by
default with only the Main and Security repositories enabled.  You can
edit your list of available repositories by editing your sources.list
file in the /etc/apt/ directory.  Before you do so for the first time,
I would reccomend backing up sources.list by opening a terminal window
and issuing the following command:
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
You can then edit your sources.list file by using the following command:
sudo kate /etc/apt/sources.list

Read the comments in the file to activate the Universe repository.
Basicly, you will be removing the pound (#) character from in front of
the following lines
# deb http://us.archive.ubuntu.com/ubuntu dapper universe
# deb-src http://us.archive.ubuntu.com/ubuntu dapper universe
# deb http://security.ubuntu.com/ubuntu dapper-security universe
# deb-src http://security.ubuntu.com/ubuntu dapper-security universe

Save the file and then update your repositorys either usuing synaptic
or the command
sudo apt-get update

You should now be able to find many more applications to install.
