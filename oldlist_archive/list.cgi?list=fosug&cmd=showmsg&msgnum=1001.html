This was a big concern of mine.  Pissed me off that the screen remained on
durring a call, (keyboard too)  eating away my battery life.  This was fixed
with three things.  The first one, was a firmware update from sprint that
allowed you to turn screen off durring calls. But from my understanding the
other carriers havn't releaed an update.  Hence a program called "Butler"
This program is an all around neato addition to the treo. IT adds 6 quick
alarms, custom ringtones, autoscreen locks,  Fixing the backlick problems.
It also adds a feature where you can SMS your phone if stolen with a keyword
and it will nuke the contents, of both phone and memory stick, in
background.  Second program is call KBlights.  This program gives you
greater control over the keyboard backlight.  I HATED turning my treo on in
a dark room.  Keyboard would cause my eyes to cower behind my brain stem.
KBlights gives you control over when, how bright,, etc.  Now I have my
keyboard come on ONLY when it expects keyboard input.  I can toggle it by
tapping my side button twice quickly tho, for when I WANT to use my keyboard
as a search light.

Checkout www.palmgear.com  and www.treocentral.com  good places. Here is a
list of applications I keep installed on phone.

Zlauncher *  - OS/SHell replacement - Makes treo look pretty, and also lets
you run applications OFF your memory card. That 28mb internal ram gets used
up quick


Trips * - Milage and Client billing Tracker - (probably not usefull to you)

Palm Express $ - Realtime INformation database Generate maps, dirving
directions. yellow pages, news, stocks etc $6 a month. Worth it if you drive
alot to new places

Mobie Office * - MS Office Clone, and document management system. ROCKS.
Read/writes MS doc formats nativly, or the more compressed palm versions.

Turbo Password * - Password safe, Keep track of all your/clients passwords.
Database is encrypted similar to kwallet (tho not directly compatable, you
can import/export cvs tho)


Beyond Contacts * - Outlook Replacement. Great Great calandar replacemnt if
you use outbreak or outbreak compatable calander (which I'm stuck using)


Mobi Pocket reader * - Ebook Reader

PalmPDF * - Palm PDF reader (the adobe one sucks ass)

Tibr Pro *- NICE ebook reader

Student Toolkit * - Math and science info, Peridic table unit convertion.,
Electronic diagrams (Cederquist would love this one)

Backup Man * - Ghost for palm, shedualed backups, restore (saved my ass a
few times)

Butler * - Treo tools, enhancements, (great program)

Card Export II * - Turns palm into a USB harddrive. OS sees it as hardware
usb drive, no special drivers needed

Filez * - NICE File manager (although the one built into zlaucher is getting
very close to comparing)

HandZip - Winzip for palm

Kblight - Enhancements for lighted keyboard. Battery saver

PTunes * - Mp3/Ogg player Does shoutcast and streaming mp3/ogg formats as
well.

Mvnc * - Older VNC viewer

PalmVnc * - Newer VNC viewer (will not view older)

PDANet * - Lets you use your palm as a modem, very fast over USB, also
supports much better bluetooth management. This does require windows. Not
linux compatable. you MUST use internal bluetooth for linux. If you use
windows this is much better optimized for speed.  I keep a copy of the
client on my sd card. So I can run card export and install client on any
machine I need to.

Pssh * - Version 2 SSH client

Whois * - NS lookup info

Mahjong * - Yes i'm a nerd

Alchemy * - fun game

GBulator $ - Does GB and GB color emulator. 2000+ games in 300mb of card
space. ;)

NesEM * - Original NEs emulator

Gizmo Ultra - Sega and game gear, master master II, mark III system emulator

Kronos - Z engine games, Like Zork

VIdeo Jigsaw - Fun

Solataire - Days of wasted fun.


Tho out of all these apps if I had to pick 3.  Would be Zlauncher, Backup
