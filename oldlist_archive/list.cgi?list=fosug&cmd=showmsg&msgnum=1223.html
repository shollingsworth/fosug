Granted a fresh install would be the best (but I don't know that it
will be any easier to diagnose why the install fails over the list),
but kubuntu-desktop is a complete package which installs all required
dependencies, and configures itself to the standard (k)ubuntu look and
feel.  I also believe (though I'm not certain) that dpkg confgures X
the same way using this process as it does durring the standard
install.  At worst, she has only lost time time it takes apt to
download and install the kde packages, and can do the reinstall if it
doesn't solve her problem.  At best, she's up and running with KDE.
At medium, if doesn't configure X correctly, and she desires to, she
can learn about how the X config file works.
