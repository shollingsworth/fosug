The desktop is stored in the profile also :)

You can right-click on the My Docs folder you can put My Docs on a
network share.  If your only supporting a few computers then you can
just setup the redirect My Docs for each of your users.  The setting
will be applied to their profile so it will roam with them.

If your on an AD domain then you can also make a GPO that redirects the
desktop folder, My documents, Start Menu, etc all to network shares.  I
just checked on some profiles sizes at work and they all look to be
about 6-10MB and the majority of this size comes from the Application
Data folder.  If I felt really ambitious I could also redirect the
Application Data folder to a network share.
