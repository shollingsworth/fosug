alias backupcurrentdir="tar -cjf ~/.backup/\`pwd|sed -e
'\''s/\//-/g'\''|sed -e '\''s/^-//'\''\`_\`date
+%Y\_%m\_%d\_%H\_%M\_%S\`.tar.bz2 `pwd`"

Just thought I'd share. Robert will most likely correct me, but I put
this alias in my .bashrc file just like shown it will make a tar file
of the directory you are in, and put it in your backup dir, in my case
'~/.backup'

Works well if you're doing a lot of text editing on the command line
and your directory is full of text.
