On 2006.01.21 22:16:46 -0800, Joel Searcy wrote:
> In the next few months my job duities are going to gradually change  here
> towards more NOC support and going out to do at location tech  support.
> I'll be doing some Cisco/networking stuff but mostly  windows servers due to
> the fact that we sell a lot of them and the're  easy to get people to buy
> contracts for.  I don't really have much  of a choice in operaiting systems
> I want to work with so I have to  except as is.  It's been quite some time
> sense I have done this on  a regular basis and I am finding I need to
> assemble a toolkit for  future use (both hardware and software).  From you
> guru guys who  do this stuff could you assemble a list of suggions for
> hardware/software for me to cary around as well as quick reference
> materials that will come in handy.  I need to be prepared for most
> situations that might arise.  Thanks in advance.

Software tools:
  o A good live cd.  Something like whax [1] or some other.  If you find that
    you need to diagnose a problem from a client machine, you can boot to a
    known state.  Also, this has pen test utils that you can use if that comes
    up.
  o The windows Free Tool Downloads page [2] is great when you need some weird
    util that you never thought you'd use.
  o Just about every utility from SysInternals [3].  Especially Autoruns [4].
    
Hardware tools:
  o A small toolset including:
      x 2 different phillips screwdrivers.  One 0 size and one that'll fit
        small screws (cannot recall the numbers - I think like a 5 or such).
      x 2 flathead screwdrivers.  Same parameters as the phillips.
      x A complete set of torx drivers (10 and 15 are *very* common).
      x A pair of tweezers.
      x A telescoping mirror. [5] 
      x A flashlight (always carry a spare set of batteries).
      x A tape measure.
  o If you can afford it and have the need, a cable tester is *very* nice.  My
    work has a fluke that does a number of tests.  The most useful of which
    are telling if the wire map is correct and if not, what's wrong and if
    there's a break, where that is.
  o If you need to trace wires, a trace kit is a must.

This is not a complete list, however this is the absolute must haves that I've
seen in doing this sort of stuff.

Good luck.
