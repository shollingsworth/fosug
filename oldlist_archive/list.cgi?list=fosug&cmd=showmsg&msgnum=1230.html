Ronnie,

Think about what you want to use it for.  I find them very handy when they
are key fob size and don't require a cable.  That way you can carry it
everywhere and always have important data available.  They are great for
backing up what you're working on and making it easy to use more than one
computer -- say work and home.  The one I've used is only 256MB and it
serves me well.  If I purchased a new one today I might be inclined to get a
1GB unit.  Circuit City has a large selection so that you can peruse them in
person.

Have a great day -- George
