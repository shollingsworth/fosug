Hi Ronnie,

Basicly, you can think of an ISO as type of zip (though without the
compression) file.  It contains the image of all the files needed to
boot an run the live cd.  In order to turn the iso into a bootable
disk (instead of a cd with one .iso file), you need to select the
appropriate option in your burning software.  For example, using Roxio
version 6 (and version 5 if memory serves) you select File | 'Record
disk from image' and then select the .iso image you wish to burn.
This will 'unzip' all the files in the .iso to the disk, and make it
bootable.

If you are unable to find this option in your cd-burning software, or
your version of the software doesn't support it (some of the 'light'
software versions are crippled in some fashion), then there are, as
Drew mentioned, utilities that you can download to handle this.  If
that doesn't work, please feel free to let me know, and I'd be happy
to run a copy of Ubuntu over to you.

By the way, the most current version of Ubuntu (6.06 LTS) was released
overnight, so you might want to download the newest ISO.  I've been
running beta versions of this release for over a month now, and in my
opinion it is an excelent piece of software.  My system actually runs
faster using version 6 than it did using version 5.

Sorry to drone on so long....
Regards,
-Chad
